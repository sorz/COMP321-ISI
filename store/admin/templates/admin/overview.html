Overview

Welcome to ISI Store admin console.

Please select an operation from left side menu.

  • Product management

    View all products by selecting a category. Show and edit detail of each product. You can also modify title and introduction of each category here.

  • Add new product

    Add a new product into ISI Store.

  • Add new category

    Add a new category into ISI Store.

  • Pending orders

    List all pending (or hold) purchase orders. You can hold or ship a purchase order here.

  • On-delivery orders

    List all purchase orders that are shipped, but customers have not confirmed receipt yet.

  • Fulfilled order report

    List the fulfilled purchase orders. i.e. purchase orders that are delivered to customers and for which customers have confirmed the receipt.

  • Cancelled order report

    List all cancelled orders.
